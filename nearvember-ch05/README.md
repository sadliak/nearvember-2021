# NEARvember #5 (Rust NFT)

## Usage

* Install dependencies
  ```shell
  $ yarn
  ```

* Compile contract to WASM and build web
  ```shell
  $ yarn build
  ```

* Deploy contract (described in root `README.md`)

* Start web
  ```shell
  $ yarn start:web
  ```

* Deploy web
  ```shell
  $ yarn deploy:web
  ```
